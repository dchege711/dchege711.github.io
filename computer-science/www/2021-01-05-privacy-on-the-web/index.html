Just Click on this Simple URL

One can use curl -v <url> to see all of the redirections and cookies set. []

I tried this on https://tinyurl.com/yyte6pqv, which points to https://www.c13u.com/socioeconomics/2020-11-21-experiments-in-social-economics/, but didn’t observe any 3rd party cookies. tinyurl.com did set a cookie, but that’s probably expected[?]

[] found that their tiny URL landed them at https://redirect.viglink.com?key=a7e37b5f6ff1de9cb410158b1013e54a&u=https%3A%2F%2Fzoom.us%2Fj%2F123456789&prodOvrd=RAC, which eventually goes to https://zoom.us/j/123456789. I manually created another tiny URL for the Zoom link and got the redirect.viglink.com URL once again. Bonkers!

https://t.ly/ uses no trackers. Their revenue model is a $5/month upgrade that allows custom domains, customizable links, expiring links, private stats and API access. []
